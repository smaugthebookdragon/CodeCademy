HTML Table Cheatsheet

Name Description Example code
<table> This tag defines an HTML table <table>
 ... Table contents ...
</table>
<tr> This tag creates a table row <tr>
 ... Table row contents ...
</tr>
<td> This tag creates a table data cell <td>Cell contents</td>
<th> This tag creates a table header <td>Header title</d>
<caption> This tag adds a caption to a table <td>Table description</d>
colspan This attribute makes a cell span more than one column <th colspan="2">Text</th>
rowspan This attribute makes a cell span more than one row <th rowspan="2">Text</th>
